# **App Name**: Drona IQ

## Core Features:

- AI Academic Diagnostic Tool: An AI-powered reasoning tool that analyzes student assessment performance to dynamically generate a personalized learning gap report and suggest a tailored study path.
- High-Conversion Lead Magnet Assessment: A multi-step 'Academic Health Checkup' form optimized for lead generation, capturing student data and weak subjects directly into the cloud database.
- Parent Success Analytics Dashboard: A premium dashboard for parents to track real-time attendance, test reports, and student growth metrics stored in Firestore.
- Competitive Excellence Matrix: An interactive UI component that visually compares Drona IQ's outcomes and features against local competitors like PW and ThinkArk to drive trust.
- AI Study Assistant: A generative AI tool that acts as a 24/7 academic mentor, using contextual reasoning to provide student support and curriculum-specific guidance.
- Local SEO Dynamic Content Engine: A high-performance system for generating SEO-ready landing pages and blogs targeting Dehradun-specific tuition and defence foundation keywords.
- Automated Admission Routing: A logic-based workflow that processes incoming inquiries and routes them to academic counselors based on student profile complexity.

## Style Guidelines:

- Primary: Deep Navy (#0B132B) for institutional authority; Accent: Regal Gold (#C5A021) for premium prestige; Neutral: Stark White (#FFFFFF) for academic clarity.
- Headlines use 'Montserrat' for a bold, modern, and trustworthy feel; Body text uses 'Lato' for high readability on reports and dashboards.
- Clean, minimalist line art icons with a subtle military-inspired aesthetic and gold-stroke highlights.
- A conversion-focused F-pattern layout with high-contrast sections designed to move parents from 'problem recognition' to 'trust building' to 'CTA'.
- Precision-based micro-interactions on data visualizations and smooth, logical reveal animations during the student assessment process.
